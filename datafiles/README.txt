Hi there! Thanks for downloading and checking out Pizza Tower: Lamb Fixes, an updated, bug-fixing version of the popular but controversial mod Cheesed Up. This mod primarily adds a lap-3 song for Noise, a few new patterns, and makes the game compatible with PO/PO+ for more general use.

I'm aware of how problematic and chaotic the history of this mod is, mainly because it stems from its maker. I'm here to say that I have no ties to or connections to Lila or anyone behind CU or any other Cheesed Up project. What started as a simple mod to fix a mod I like and to add a few new things turned into a fun project for a week of College.

If you like what you play, leave a review and comment. This is meant to be a small project to help me understand Pizza Tower's engine and code so I can make something else. Have fun!
